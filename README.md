# SBT packager #

This is an SBT plugin for software packaging. It depends on [sbt-native-packager](https://raw.github.com/sbt/sbt-native-packager).

## Installation ##

    addSbtPlugin("com.github.malliina" % "sbt-packager" % "0.9.9")
